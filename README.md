# rs-hash-table
Implementation of hash table with Rust. Using open addressing for handling collision and djb2 as the hash function.


> ⚠️ Only for educational purposes. Use use `std::collections::HashMap`

# Quick Start

Run with cargo

```terminal
cargo run 
```
